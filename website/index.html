DevLauncher

一键启动你的开发工作流

虚拟键盘是 DevLauncher 的主入口

按键就是动作。常用工作流绑定一次，之后用肌肉记忆直接触发。

⌃ Summon

快捷唤起

macOS 下双击 Control 快速打开或隐藏虚拟键盘，也可以使用 Cmd+Opt+J 作为备用组合键。

⌘ Bind

点击绑定

点击空键位，选择动作类型，填写目标后保存。已绑定键位可以再次编辑或拖拽交换位置。

▦ Actions

覆盖高频动作

支持应用、文件夹、文件、网址、SSH、脚本、系统命令、内置功能和 marketplace 插件动作。

应用程序 文件夹 文件 网址 SSH 脚本 系统 内置 插件

极速连贯的核心工作流

从入口、动作到插件沉淀，都围绕 DevLauncher 现有桌面能力展开。

⌁
Step 01

召唤入口

通过桌面悬浮宠物、全局快捷键或虚拟键盘，瞬间唤醒你的工作流控制中心。

⌕
Step 02

选择内置工具

打开剪切板历史、截图、JSON 助手、快捷记忆、终端、TOTP 或远程桌面等真实内置功能。

▰
Step 03

运行本地动作

调用 Tauri 本地能力、终端命令、SSH 会话或静态 WebView 插件，不打断当前思路。

◇
Step 04

沉淀为插件/快捷键

将高频操作固化到 keyboard.yaml、自定义宠物菜单或 marketplace 插件中，持续进化。

全能的核心能力矩阵

来自项目内置 manifest 和已打包 marketplace 插件的真实能力。

♟

桌面宠物入口

支持点击展开、固定层级、自定义菜单，以及 Codex 状态联动。

▤

剪切板历史

打开历史记录窗口，搜索复制内容并一键粘贴。

⌗

截图与问题报告

全屏截图、区域选择、标注，并把截图整理成 AI Prompt 问题报告。

{ }

JSON 助手

格式化、转义/去转义 JSON，并生成 OpenAI 文档片段。

MEM

快捷记忆

沉淀开发常用命令与快捷键，作为本地速查面板使用。

▰

终端与 SSH

内置终端窗口，支持命令执行与 SSH 会话。

▣

远程桌面

RDP 管理与 P2P 屏幕共享入口集中管理。

WEB

网页账号

管理 Chrome 网页账号绑定、扩展安装和安全填充说明。

▦

插件市场

已打包 API Lab、Developer Toolkit、Writing & Code Toolkit 等 WebView 插件。

▦Extensible Architecture

插件不是附属品，
而是工作流的延伸

项目已经内置静态 WebView 插件市场与 release zip 包。你可以用 HTML、CSS、JS 编写工具界面，打包后通过 marketplace 分发到 DevLauncher。

⌘

API Lab

轻量 API 请求工具，支持环境、集合、历史与 JSON 响应查看。

▧

Developer Toolkit

时间戳、编码、哈希、正则、Cron 与二维码等开发小工具集合。

☷

Writing & Code Toolkit

Diff、Markdown 预览、代码截图与 snippets，服务写作和代码整理。

Local-First

▥

数据存留本机

keyboard.yaml、插件包、截图记录与工具状态优先保存在本地应用目录，敏感凭据交给系统 Keychain/Keyring。

◈

桌面原生响应

基于 Tauri 调用本地能力，工具窗口可置顶、可独立打开，适合高频小任务。

⬟

完全掌控

配置、插件与文档都在仓库里可追溯，适合按自己的桌面习惯持续打磨。

为真实的极客场景而生

▱

启动项目环境

通过虚拟键盘绑定文件夹、URL、App 或脚本，一次打开项目相关入口。

⌕

找回复制记录

呼出剪切板历史，搜索刚复制过的报错、命令或链接，回车直接粘贴。

⚡

快速调试 API

打开 API Lab，管理环境和集合，复用历史请求并查看响应。

✕

整理截图问题

先截图和标注，再用截图问题报告窗口整理上下文，生成可复制的 AI Prompt。

▰

终端与远程连接

打开内置终端执行本地命令，或进入 SSH、RDP、P2P 屏幕共享相关工作流。

{ }

处理临时文本

用 JSON 助手和 Developer Toolkit 处理格式化、编码、哈希、时间戳和正则。
