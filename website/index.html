Core workflow

不是快捷方式列表，而是一张可记忆的工作台。

你可以按开发、运维、工具等场景创建多页键盘，拖拽换位、右键编辑，并在窗口隐藏时继续用全局快捷键触发。

多页面键盘

按不同工作场景分组，Tab 或 Shift + Tab 快速切换。

多动作绑定

支持应用、文件夹、文件、URL、SSH、脚本、系统命令和内置功能。

全局快捷键

窗口隐藏时仍可用 Alt + Space 呼出，用 Alt + 键位触发当前页动作。

主题设置

背景、透明度、模糊、边框和按键透明度都可以按偏好调整。

Make a plugin

新增插件时，接入现有 registry。

内置功能位于 src/builtins/。新增插件时创建 manifest 和独立 App，再注册到 BUILTIN_REGISTRY，最后补上 Tauri 窗口配置和 Rust 命令入口。

  1. 01创建 src/builtins/<id>/manifest.ts
  2. 02创建 src/builtins/<id>/App.tsx
  3. 03在 BUILTIN_REGISTRY 中注册 manifest 和组件
  4. 04在 tauri.conf.json 添加插件窗口配置
  5. 05在 Rust 中添加 toggle_<id>_window 命令
