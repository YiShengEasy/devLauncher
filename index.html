Core workflow

主页介绍围绕项目真实能力展开。

DevLauncher 的核心不是普通官网模板，而是一个可配置的开发者启动器：多页面键盘、拖拽换位、 右键编辑、全局快捷键、托盘隐藏和本地配置持久化。

多页面键盘

按开发、运维、工具等场景创建多个键盘页，Tab 或 Shift+Tab 切换。

多动作绑定

支持 App、文件夹、文件、URL、SSH、脚本、系统命令和内置功能。

全局快捷键

窗口隐藏时仍可用 Alt+Space 呼出，用 Alt+按键触发当前页绑定。

主题设置

玻璃面板支持背景、透明度、模糊强度、边框颜色和按键透明度配置。

Make a plugin

新增插件时，按项目现有 registry 接入。

当前项目的内置功能位于 `src/builtins/`。新增插件时创建 manifest 和独立 App， 再注册到 `BUILTIN_REGISTRY`，最后在 Tauri 配置和 Rust 命令里补上窗口入口。

  1. 01创建 `src/builtins/<id>/manifest.ts`
  2. 02创建 `src/builtins/<id>/App.tsx`
  3. 03在 `BUILTIN_REGISTRY` 中注册 manifest 和组件
  4. 04在 `tauri.conf.json` 添加插件窗口配置
  5. 05在 Rust 中添加 `toggle_<id>_window` 命令
