# DevLauncher

> 面向开发者和高级用户的桌面效率启动器

## 一、项目定位

DevLauncher 是一个常驻后台的桌面效率工具。

用户通过快捷键呼出一个虚拟键盘面板，在每个按键上绑定不同动作，实现快速启动应用、打开项目、连接服务器、执行脚本等操作。

产品定位介于：

* Quicker
* Raycast
* Alfred
* Stream Deck

之间。

核心目标：

> 用最少的鼠标点击完成最常用的操作。

# 五、核心概念

## Action

系统中的所有功能统一抽象为 Action。

例如：

* 打开应用
* SSH连接
* 打开网址
* 执行脚本

本质上都是 Action。

---

## Page

一个虚拟键盘页面。

例如：

```text
开发
运维
办公
AI
```

用户可以切换页面。

---

## Key

虚拟键盘中的单个按键。

例如：

```text
Q
W
E
R
A
S
D
F
```

每个键位可绑定一个 Action。

# 七、配置示例

keyboard.yaml

```yaml
pages:
  - name: 开发

    keys:

      Q:
        type: app
        name: VSCode
        target: C:\Program Files\Microsoft VS Code\Code.exe

      W:
        type: ssh
        name: 测试环境
        host: 192.168.1.100
        user: root

      E:
        type: folder
        name: 项目目录
        target: D:\Project
```

# 十一、成功标准

V1发布后用户可以：

* 不打开桌面图标
* 不打开开始菜单
* 不打开资源管理器

仅通过：

```text
Alt + Space
```

即可完成：

* 启动应用
* 打开项目
* SSH连接
* 执行脚本

实现开发工作流的快速入口。

最终目标：

成为开发者桌面环境中的统一入口（Unified Workspace Launcher）。
