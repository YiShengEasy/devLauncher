iOS App Logo Set

Folders:
- AppIcon.appiconset: Xcode-ready icon set. Drag this folder into Assets.xcassets or replace existing AppIcon.appiconset.
- by_size_png: PNG files named by final pixel size, such as 1024x1024.png, 180x180.png, 120x120.png.
- logo-master-1024x1024.png: master square logo.

Note: iOS app icons should be square PNGs without transparency. These files are RGB PNGs.
